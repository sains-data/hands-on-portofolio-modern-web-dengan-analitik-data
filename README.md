# 📘 Hands On Portofolio Modern Web dengan Analitik Data: Dari Coding ke Insight

Repositori ini merupakan pusat dokumentasi dan kode sumber untuk proyek buku **"Hands On Portofolio Modern Web dengan Analitik Data"**. Proyek ini dirancang sebagai platform pembelajaran integratif yang menggabungkan pengembangan web *full-stack*, analitik data, dan implementasi kecerdasan buatan.

## 📖 Tentang Buku
Buku ini disusun sebagai jawaban atas kebutuhan literatur pengembangan web yang praktis, modern, dan mudah dipahami oleh khalayak luas. Dengan menggunakan pendekatan **Learning by Doing**, pembaca akan dipandu langkah demi langkah untuk membangun ekosistem portofolio digital yang profesional. Fokus utama buku ini adalah menjembatani teori teknis dengan implementasi nyata, sehingga pembaca tidak hanya mahir menulis kode, tetapi juga mampu mengelola infrastruktur dan mendapatkan wawasan berharga dari data.

---

## 📝 Pendahuluan
Perkembangan teknologi web telah mengubah cara profesional menyajikan informasi dan wawasan data. Buku ini memberikan respons terhadap kebutuhan tersebut dengan menyajikan panduan membangun sistem yang terintegrasi dengan analitik data. Melalui pendekatan full-stack, pembaca akan memahami keseluruhan siklus pengembangan aplikasi, mulai dari pengelolaan infrastruktur cloud, pengembangan backend yang aman, hingga pembuatan dashboard front-end yang interaktif dengan dukungan teknologi OpenAI.

## 🎯 Output yang Diharapkan
Melalui praktikum yang ada di repositori ini, mahasiswa diharapkan mampu:
1. **Menguasai Siklus Full-stack:** Memahami alur kerja pengembangan web dari perencanaan server hingga aplikasi siap pakai.
2. **Implementasi AI & Analytics:** Mampu mengintegrasikan layanan OpenAI dan pelacakan perilaku pengguna menggunakan PostHog.
3. **Kesiapan Kerja:** Membangun keterampilan teknis menggunakan teknologi populer seperti React, Node.js, dan MySQL yang relevan dengan kebutuhan dunia kerja saat ini.
4. **Portofolio Digital:** Menghasilkan produk akhir berupa platform portofolio profesional yang terdokumentasi dengan baik.

## 🛠️ Stack Teknologi
* **Frontend:** ReactJS, Tailwind CSS, Vite
* **Backend:** Node.js, Express.js
* **Database:** MySQL
* **Analitik:** PostHog
* **AI Integration:** OpenAI API
* **Server:** Linux (Ubuntu), VM IDCloud
